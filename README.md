# NotifierBot
Этот бот предназначен для того, чтобы уведомлять пользователей о выходе новых серий, отслеживаемых ими сериалов или аниме.

Немного про структуру проекта:

main.py - это главный скрипт проекта, собственно отсюда и запускается NotifierBot.

config.py - здесь мы вытягиваем токен нашего бота из файла .env, который игнорируется git'ом.

loader.py - в этом скрипте мы инициализируем нашего бота и диспетчер.


db - в этой директории хранятся скрипты для работы с БД и сама БД.

    db_api.py - здесь описан родительский класс для работы с БД.
    
    db_api_anime.py - в этом скрипте производится работа с таблицей в БД anime_titles.
    
    db_api_serial.py - в этом скрипте производится работа с таблицей в БД serial_titles.
    
    notifier.db - сама БД.
    
    
handlers - в этой директории хранятся обработчики команд.

    anime.py - в этом скрипте описана обработка команда /add_anime - которая добавляет аниме в список отслеживаемого.
    
    general.py - в этом скрипте обрабатываются общие команды /info - для получения информации о сериалах и аниме, которые отслеживает пользователь, /delete - команда
которая удаляет аниме или сериал из списка отслеживаемого, /leave - команда, которая удаляет пользователя и все данные о нем из БД.

    intro.py - в этом скрипте обратваются начальные команды /start - которая добавляет нового пользователя в БД, /help - которая выводит список команд для пользователя.
    
    serial.py - в этом скрипте обрабатывается команда /add_serial - которая добавляет сериал в список отслеживаемого.
    
    
parser - в этой директории хранится парсер сериалов и аниме.

    init_parser.py - скрипт, в котором хранится родительский класс и инициализируется драйвер selenium'а.
    
    anime.py - скрипт, в котором хранится парсер для сбора данных об определенном тайтле аниме.
    
    serials.py - скрипт, в котором хранится парсер для сбора данных об определенном тайтле сериала.
    
    chromedriver.exe - драйвер хрома, для работы selenium'а.
    
    
tasks - в этой директории хранятся задания, которые наш бот выполняет каждый день.
    anime_check.py - здесь мы проверяем выходит ли сегодня новая серия, какого-либо аниме.
    serial_check.py - здесь мы проверяем выходит ли сегодня новая серия, какого-либо сериала.
    send_msg.py - здесь мы отправляем определенным пользователям уведомление о том, что вышла новая серия отслеживаемого аниме/сериала.
